This is a script to take a debian box from ground zero to Rails Testing Machine. You will end up with MySQL, Postgres, Sqlite, Memcached and many rubies, as well as automated testing scripts.

install instructions:

1) Move the <distro>_setup.sh file to your work account's home directory
2) Setup password-less sudo for your work account
3) as your work account, chmod 700 ~/<distro>_setup.sh
4) run <distro>_setup.sh
